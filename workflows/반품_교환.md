# 반품/교환 워크플로우

## 분류 기준
반품 신청, 교환 요청, 환불 처리, 반품 기간, 반품 불가 사유 관련 문의

## 핵심 정보 추출
답변 전 다음 정보를 파악하세요:
- **요청 유형**: 반품 / 교환 / 환불
- **사유**: 단순 변심 / 불량·오배송 / 기타
- **구매일 또는 수령일** (언급된 경우)
- **상품 상태**: 미개봉 / 사용함 / 파손

## 답변 구조
1. 요청 유형과 사유 확인
2. 해당 케이스의 처리 가능 여부 및 조건 안내
3. 반품/교환 신청 방법 단계별 안내
4. 환불 예상 시점 안내

## 답변 규칙
- 단순 변심 반품: 수령 후 7일 이내, 배송비 고객 부담
- 불량·오배송: 수령 후 3개월 이내 또는 인지 후 30일 이내, 배송비 판매자 부담
- 반품 신청 경로: 스마트스토어 앱 > 구매내역 > 반품/교환 신청
- 교환은 동일 상품만 가능하며 재고 없을 경우 환불 처리됨을 안내
- 반품 불가 조건 (사용·훼손·라벨 제거 등) 해당 시 명확히 안내

- "환불은 언제 되나요?"
