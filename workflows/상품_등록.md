# 상품 등록 워크플로우

## 분류 기준
상품 등록, 상품 수정, 옵션 설정, 재고 관리, 카테고리 분류, 상품 이미지 관련 문의

## 핵심 정보 추출
답변 전 다음 정보를 파악하세요:
- **문의 유형**: 신규 등록 / 수정 / 옵션 설정 / 재고 / 이미지 / 카테고리
- **문제 상황** (언급된 경우): 오류 메시지 / 반려 / 노출 안 됨

## 답변 구조
1. 상품 등록 관련 문의 유형 확인
2. 해당 기능의 위치 및 처리 방법 단계별 안내
3. 주의사항 또는 정책 안내

## 답변 규칙
- 상품 등록 경로: 스마트스토어 센터 > 상품 관리 > 상품 등록
- 상품 반려 시 → 반려 사유 확인 후 수정 재등록 안내
- 금지 품목(의약품, 성인용품 등) 등록 시도 시 정책 위반 안내
- 이미지 규격: 권장 1000×1000px 이상, jpg/png 형식
- 옵션은 최대 3단계까지 설정 가능함을 안내
- 상품 노출이 안 될 경우 검수 중일 수 있음 안내 (최대 영업일 1~2일)

## 후속 질문 예시

- "상품이 검색에 안 나와요"
- "이미지 등록 규격이 궁금해요"
