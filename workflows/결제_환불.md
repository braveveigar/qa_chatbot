# 결제/환불 워크플로우

## 분류 기준
결제 수단, 결제 오류, 할부, 포인트·쿠폰 사용, 환불 금액·시점 관련 문의

## 핵심 정보 추출
답변 전 다음 정보를 파악하세요:
- **문의 유형**: 결제 실패 / 결제 수단 문의 / 포인트·쿠폰 / 환불 금액 / 환불 지연
- **결제 수단** (언급된 경우): 카드 / 무통장 / 네이버페이 / 기타
- **환불 요청 여부**

## 답변 구조
1. 결제 또는 환불 문의 유형 확인
2. 해당 수단 또는 상황에 맞는 처리 방법 안내
3. 예상 처리 기간 안내

## 답변 규칙
- 환불 시점: 카드 3~5 영업일 / 무통장 1~3 영업일 / 네이버페이 즉시~1 영업일
- 결제 실패 시 → 카드사 또는 PG사 문의 안내
- 포인트·쿠폰은 반품 완료 후 자동 복구됨을 안내
- 부분 환불(일부 상품 반품)은 해당 상품 금액만 환불됨을 명시
- 환불 계좌는 주문 시 등록된 계좌로 처리됨을 안내

## 후속 질문 예시
- "환불은 언제 들어오나요?"
- "네이버페이로 결제했는데 환불 방법이 궁금해요"
- "쿠폰이 사라졌어요"
